# 🛡 MotionGuard – Intelligent CCTV Motion Detection System

An AI-powered surveillance system that detects motion using spatial and temporal feature learning through a CNN–BiLSTM architecture.

---

## 🚀 Project Overview

MotionGuard enhances traditional CCTV systems by applying deep learning techniques to detect abnormal motion patterns in video sequences.

It combines:

- Convolutional Neural Networks (CNN) for spatial feature extraction  
- Bidirectional LSTM (BiLSTM) for temporal pattern recognition  

---

## 🧠 Model Architecture

CNN → Feature Extraction  
BiLSTM → Sequence Learning  
Fully Connected Layer → Motion Classification
